This use case applies when the 'Solutions - Subcategory Page' loads.

#### Notes
Use one of the following values for the `page.gaStep` variable based on the scenario that may apply.

ga_step=mobile_solutions_solutions_ + [subcategory name]

Here multiple variations:

* ga_step=mobile_solutions_solutions_retail
* ga_step=mobile_solutions_solutions_route_accounting
* ga_step=mobile_solutions_solutions_transportation
* ga_step=mobile_solutions_solutions_public_safety
* ga_step=mobile_solutions_solutions_clinical and home healthcare
* ga_step=mobile_solutions_solutions_datacom
